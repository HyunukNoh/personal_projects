# Lyric Learner

## Setup

### 1. Install dependencies
```
npm install
```

### 2. Load songs
```
npm run songs
```
This clears `assets/songs`, copies all songs from `pipeline_output/songs_in_app` (using snake_case names derived from `song_info.json`), then regenerates `songs_manifest.json` and `songs_audio.ts`. Run this any time the pipeline output changes.

### 3. Run on device
```
npm run ios
```

## Notes

- `scripts/` and `assets/songs` are gitignored — they are local to your machine.
- `src/data/songs_manifest.json` and `src/data/songs_audio.ts` are also gitignored (generated by `npm run songs`).
- The pipeline output folder is expected at `../pipeline_output/songs_in_app` relative to this project.
